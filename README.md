# InAmigos Foundation Website

A responsive single-page website developed for **InAmigos Foundation** as part of a Web Development Internship task.

## Live Demo

https://dhaarani-pm.github.io/InAmigos-foundation-AWARENESS-WEBPAGE/

## Features

- Responsive design
- Modern landing page
- About section
- Areas of Impact
- Interactive project cards
- Gallery
- Volunteer section
- Donation page
- Contact section
- Footer with social links

## Technologies Used

- HTML5
- CSS3
- Google Fonts
- Font Awesome

## Project Structure

```
├── index.html
├── style.css
├── imgs/
└── README.md
```

## Purpose

This website was developed to raise awareness about the initiatives of InAmigos Foundation and provide visitors with information about its mission, projects, volunteering opportunities, and donation platform.
